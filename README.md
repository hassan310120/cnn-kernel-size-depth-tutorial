# Effect of Kernel Size and Network Depth on CNN Performance

This repository contains the code and tutorial for an individual assignment
investigating how kernel size and network depth affect the performance of
Convolutional Neural Networks (CNNs) on the CIFAR-10 dataset.

## Contents
- `cnn_cifar_experiment.ipynb`: notebook containing the full runnable code
- `tutorial.pdf`: Step-by-step tutorial explaining the methodology, experiments, and results
